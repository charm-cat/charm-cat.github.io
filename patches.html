Universal patches are more general-purpose, and may not be as reliable as app-specific patches.
You may face issues using them. To use it anyway, turn on "Allow using universal patches" in Advanced settings.
Disable Pairip license check
Disables Play Integrity API (Pairip) client-side license check.

Spoofs an existing Wi-Fi connection.
Prevent screenshot detection
Removes the registration of all screen capture callbacks. This prevents the app from detecting screenshots.

Override certificate pinning
Overrides certificate pinning, allowing to inspect traffic via a proxy.
Remove screen capture restriction
Removes the restriction of capturing audio from apps that normally wouldn't allow it.
Set target SDK version 34
Changes the target SDK to version 34 (Android 14). For devices running Android 15+, this will disable edge-to-edge display.
Enable Android debugging

This can slow down the app.

This will turn off app store updates and allows downgrading an existing app install to an older app version.

Spoofs information about the SIM card provider.

Remove share targets

Spoof root of trust

Spoof build info(Configurable)

Hide ADB status

Export internal data documents provider

Export all activities
